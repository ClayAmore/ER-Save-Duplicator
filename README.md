# WARNING
This will duplicate your save for every account that has an Elden Ring save. This means that if you have a family member/significant other that plays on their own account then this will also duplicate their save file. It will however make a backup of that save file before overwriting. The backup will be name `Sl0000.<The extenison you picked>.backup`.
